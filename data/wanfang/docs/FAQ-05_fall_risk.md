| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「高危跌倒評估」畫面後沒有資料？ | 需先選擇護理站與病人，再進入「護理評估 → 高危跌倒評估」功能（第 4 頁）。 |
| 為什麼無法新增跌倒傾向評估？ | 請先於功能區選擇【護理評估 → 跌倒傾向評估】，再按【新增跌倒傾向評估表】進入填寫頁面（第 5 頁）。 |
| 填寫完成後系統沒有儲存？ | 新增評估完成後必須按【存檔並離開】；若按【放棄】，輸入資料不會被儲存（第 5 頁）。 |
| 修改後的評估資料沒有更新？ | 需於文件清單中點【編輯】，修改後按【存檔並離開】，系統會顯示「修改成功」訊息（第 6 頁）。 |
| 想查看過往評估紀錄要怎麼做？ | 於主選單按【護理評估 → 跌倒傾向評估】，系統會自動載入所有文件清單（第 7 頁）。 |
| 刪除評估後資料仍存在？ | 按【刪】後，需在跳出的確認視窗中選【確定】才能完成刪除（第 8 頁）。 |
| 無法預覽或列印跌倒傾向評估表？ | 需於清單中按【預覽列印評估表】，確認瀏覽器允許彈出視窗後方可顯示內容（第 8–9 頁）。 |
| 為什麼沒有顯示「預防病人跌倒措施查檢表」？ | 只有當跌倒傾向評估總分 ≥6 分時，系統才會自動導入查檢表功能（第 10 頁）。 |
| 想新增跌倒預防查檢表要怎麼做？ | 在跌倒傾向評估總分 ≥6 的記錄中按【新增查檢表】，輸入內容後按【存檔並離開】完成新增（第 11 頁）。 |
| 修改查檢表後沒有更新？ | 於查檢表列表中選【編輯】，修改後按【存檔並離開】，系統會顯示成功訊息（第 13–14 頁）。 |
| 刪除查檢表時出現確認視窗？ | 點【刪】後需按【確定】才能真正刪除記錄（第 17 頁）。 |
| 為什麼刪除後還看到舊的查檢表？ | 系統需重新整理清單後才會更新畫面，可重新進入模組查詢（第 17 頁）。 |
| 想預覽或列印查檢表要怎麼操作？ | 於查檢表清單按【預覽列印查檢表】，系統會開啟新視窗顯示預覽畫面，再按【列印】輸出（第 18–19 頁）。 |
| 預覽列印畫面沒有跳出？ | 請確認瀏覽器未封鎖彈出視窗（第 19 頁）。 |
| 存檔後系統自動跳到查檢表頁面？ | 當跌倒傾向評估結果 ≥6 分時，系統會自動導向新增預防措施查檢表畫面（第 20 頁）。 |
| 想放棄編輯不存檔可以嗎？ | 在評估畫面按【放棄】，系統會關閉視窗並放棄所有輸入內容（第 20 頁）。 |
| 為什麼無法輸入前一天的日期？ | 每日上午 10 點前可輸入前一日資料，10 點後僅能輸入當日資料（第 3 頁）。 |
| 想補輸前幾天的資料怎麼做？ | 需由護理主管勾選【當機補輸資料】，才可輸入系統時間前的資料（第 3 頁）。 |
