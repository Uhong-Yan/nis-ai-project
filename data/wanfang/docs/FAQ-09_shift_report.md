| 常見問題 | 解決方法 |
|---|---|
| 為什麼無法輸入前一天的交班時間？ | 系統預設記錄時間為系統時間。每日 10 點前可輸入前一日時間；10 點後不能大於系統時間，亦不能小於系統時間前 12 小時（第 3 頁）。 |
| 想補輸前幾天的交班單怎麼做？ | 需由護理主管或有權限者勾選【當機補輸資料】核取方塊，才能輸入系統時間前的資料（第 3 頁）。 |
| 為什麼我沒有「當機補輸資料」選項？ | 一般使用者沒有權限，必須由護理長職級以上授權後才能啟用（第 3 頁）。 |
| 新增交班單時沒有反應？ | 需先於版本清單中選擇「交班單」後，系統才會開啟填寫頁面（第 4 頁）。 |
| 按下【完成交班】後畫面卡住或沒存成功？ | 系統完成新增後會跳出訊息頁面提示存檔狀況；確認後可選【預覽列印】或【結束】（第 5 頁）。 |
| 存檔後要怎麼確認資料是否保存？ | 可於訊息頁面選【預覽列印】，或在版本清單查看該份文件狀態是否為「正式」或「暫存」（第 5、9 頁）。 |
| 要修改交班單要怎麼做？ | 在版本清單中選擇欲修改的交班單，按【編輯】鈕進入修改畫面（第 6 頁）。 |
| 修改交班內容後怎麼知道改了哪裡？ | 系統會以紅色框標示使用者修改部分，方便確認變更（第 6 頁）。 |
| 為什麼預覽／列印按鈕灰色不能用？ | 交班單必須是「完成」狀態才可預覽或列印（第 8 頁）。 |
| 點【預覽】後畫面沒顯示內容？ | 系統會開新視窗顯示預覽內容，請確認瀏覽器允許彈出視窗（第 8 頁）。 |
| 想暫時中斷填寫交班單怎麼做？ | 可按【暫存】或【暫存並離開】；前者保存並停留畫面，後者保存並關閉視窗（第 9 頁）。 |
| 填寫到一半不想存了可以取消嗎？ | 可按【放棄】，系統會關閉填寫視窗並放棄目前輸入內容（第 9 頁）。 |
| 「預覽」與「列印」差在哪？ | 【預覽】以文字模式顯示內容；【列印】可輸出成紙本或 PDF（第 9 頁）。 |
| 為什麼交班單內容沒更新？ | 若僅暫存未按【完成交班】，文件狀態仍為暫存，不會視為正式紀錄（第 9 頁）。 |
| 系統自動登出導致交班內容遺失？ | 若未按【暫存】或【完成交班】，資料不會保存。建議定期暫存以防閒置登出（第 9 頁）。 |

---

## 補充說明
- FAQ 依《交班單操作手冊》頁碼整理。  
- 交班單僅在【完成交班】後才會成為正式紀錄，暫存不列入統計。  
- 若畫面或按鈕無反應，請確認網路穩定並允許瀏覽器彈出視窗。  
- 當機補輸需主管權限，護理長級以上帳號才能啟用。
