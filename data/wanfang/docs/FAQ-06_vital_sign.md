| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「生理監測」後沒有資料？ | 需先選擇護理站與病人，再進入【生理監測】功能；預設畫面為「Vital Sign」（第 2–3 頁）。 |
| 輸入體溫、脈搏、呼吸、血壓時要全部填完才能存檔嗎？ | 不需要，只需填寫欲新增的測量項目即可存檔（第 2 頁）。 |
| 為什麼測量時間都一樣？ | 系統預設「體溫」時間會同步套用至「脈搏、呼吸、血壓」（第 3 頁）。 |
| 想登錄「測不到」時怎麼輸入？ | 選擇「測不到」，系統會自動帶出 CPR 片語，可自行修改後存檔（第 3 頁）。 |
| 測量值超出範圍無法輸入？ | 可在空白欄手動輸入，按【確定】後將資料帶回（第 4 頁）。 |
| 為什麼異常數值會變紅？ | 系統自動偵測異常；紅色代表過高、藍色代表過低，並開啟「護理處置清單」供選擇（第 4–5 頁）。 |
| 輸入發燒後給藥資料時要填哪些欄位？ | 必填「藥物/途徑」、「數量」、「單位」，否則無法存檔（第 6 頁）。 |
| 體溫單上為什麼有紅色空心圓？ | 若勾選「因前次處置產生的記錄」，該筆體溫以紅色空心圓表示並連線顯示（第 7 頁）。 |
| 氧氣療法或呼吸器要在哪裡輸入？ | 在「呼吸」評估中選【氧氣療法】或【呼吸器】，系統會開啟填寫視窗（第 8–9 頁）。 |
| 為什麼排便時間顯示前一天？ | 系統預設排便測量時間為「系統時間 − 24 小時」（第 9 頁）。 |
| 疼痛評估沒有存檔成功？ | 疼痛量表選擇非「不評估」時，所有欄位都需填完才能存檔（第 10 頁）。 |
| I/O 記錄時要填哪些欄位？ | 每筆需填「類型」、「數量」；特定項目還需填藥囑、部位、顏色、性質等（第 11–13 頁）。 |
| 輸入 I/O 後沒看到加總？ | 系統會自動針對相同類型與單位的項目加總（第 14 頁）。 |
| 想刪除錯誤的 I/O 資料？ | 選取項目後按【刪除】，再按【I/O 新增存檔】確認刪除（第 15 頁）。 |
| Glasgow Coma Scale 總分不對？ | 系統依病人年齡自動判斷成人或兒童版本並加總（第 17 頁）。 |
| BW／CVP／SPO₂ 為什麼看不到？ | 系統預設隱藏，需手動展開要輸入的項目（第 19 頁）。 |
| CVP／SPO₂ 出現異常時有什麼提示？ | 系統會自動開啟對應「體液過量／缺失處置清單」供選擇（第 20 頁）。 |
| 查詢更新後怎麼修改多筆資料？ | 點【更新】可整批修正該時間所有紀錄，完成後按【護理事件更新存檔】（第 23 頁）。 |
| 為什麼查詢結果有紅藍字或符號？ | 紅＝高值；藍＝低值；「＊」＝特殊處置；「@」＝已刪除紀錄（第 23–24 頁）。 |
| 查不到三班 I/O 紀錄？ | 查詢日期預設當日，可修改日期後按【查詢】，再從明細中檢視或轉入護理記錄（第 34–35 頁）。 |
| 密切量測與一般 Vital Sign 差別？ | 密切量測可設定頻率與次數（15–60 分鐘），自動產生時間表，可選擇是否顯示於體溫單（第 36 頁）。 |
| 管路管理中「名稱」為什麼自動生成？ | 系統依「種類＋部位＋位置＋編號」自動產生完整名稱與護理記錄字串（第 39 頁）。 |
| 拔管後無法再記錄引流量？ | 拔管存檔後系統即鎖定該筆管路，不可再登錄引流資料（第 41 頁）。 |
| 「特殊事件」裡的 Delivered at 為什麼只出現一次？ | 病患一次住院只能登錄一次 Delivered at，存檔後選項會隱藏（第 43 頁）。 |
| 為什麼無法輸入未來時間？ | 所有記錄時間皆不得大於系統時間（第 47 頁）。 |
| 想補輸資料卻顯示權限不足？ | 只有護理長或主管可勾選【當機補輸資料】新增過期資料（第 47 頁）。 |
| 為什麼逾期資料不能新增？ | 系統設有限制，需由有權限者授權勾選【當機補輸資料】（第 47 頁）。 |
