| 常見問題 | 解決方法 |
|---|---|
| 為什麼登入後點「管理作業」沒有反應？ | 登入後需確認帳號具有管理者權限，才能開啟「管理作業」主畫面（第 3 頁）。 |
| 點進「片語維護」後畫面空白？ | 需先選擇左側目錄項目，系統才會顯示右側清單與功能按鈕（第 3 頁）。 |
| 新增片語後沒有出現在清單？ | 需按【完成】（非暫存）後片語才會顯示並可使用；暫存狀態無法被選取（第 5 頁）。 |
| 「暫存」與「完成」差在哪？ | 「暫存」僅暫時保存但不可用；「完成」代表正式啟用（第 5 頁）。 |
| 修改片語內容後沒更新？ | 編輯後需按【完成】存檔，否則仍為舊版本；紅框代表已修改欄位（第 6–7 頁）。 |
| 刪除片語沒成功？ | 刪除後會跳出確認視窗，需按【確定】才會真正刪除（第 8 頁）。 |
| 如何建立新的片語分類？ | 點【建立新的目錄】，輸入名稱與說明後按【確定】，新分類即出現在左側清單（第 8 頁）。 |
| 想更改目錄名稱或位置？ | 使用【目錄重新命名】修改名稱，【移動目錄位置】可調整階層（第 9–10 頁）。 |
| 為什麼刪除目錄時整個分類都不見？ | 系統會連同子目錄與底下所有片語一併刪除（第 11–12 頁）。 |
| 「待評值目標查詢」查不到資料？ | 需選擇正確病房並按【查詢】；若無待評值目標，列表會顯示空白（第 13 頁）。 |
| 「特殊治療項目維護」新增後沒出現？ | 輸入名稱與編碼後必須按【特殊治療項目新增】存檔，系統才會更新列表（第 14 頁）。 |
| 想刪除特殊治療項目但刪除鍵沒反應？ | 點【刪除】後會出現確認視窗，需按【確定】才會刪除（第 15 頁）。 |
| 「自行新增衛教項目查詢」頁面沒資料？ | 僅顯示病房中自行建立的衛教項目，若無此類資料會顯示空白（第 16 頁）。 |
| 「異常處置項目維護」要怎麼新增？ | 開啟項目內容列表後，在輸入欄輸入措施編碼並按【異常處置項目存檔】（第 17 頁）。 |
| 為什麼異常處置項目無法刪除？ | 需先勾選欲刪除項目，再按【異常處置項目存檔】並在確認視窗中選【確定】（第 18 頁）。 |
| 「病房待執行記錄」查不到資料？ | 需設定查詢區間與護理站後按【查詢】，系統才會列出未完成醫囑（第 19 頁）。 |
| 查詢報表時畫面沒反應？ | 查詢前須依序選擇【護理站】→【報表種類】→【日期區間】→【查詢報表】，否則不顯示內容（第 19–20 頁）。 |
| 為什麼報表無法下載？ | 僅支援 Excel 格式下載；請確認瀏覽器允許下載並選擇儲存位置（第 20 頁）。 |
| 「管路報表」查詢條件要怎麼設定？ | 需選擇「明細」或「彙總」類型及時間區間後按【查詢報表】（第 21 頁）。 |
| 「疾病嚴重度報表」顯示空白？ | 請確認已選擇護理站與日期區間；查詢後可列印或下載（第 22 頁）。 |
| 「病房日誌報表」與「病房月報表」差在哪？ | 日誌顯示單日動態；月報顯示整月佔床率與護理時數統計（第 23–24 頁）。 |
| 「護理計畫報表」有哪些種類？ | 包含健康問題、護理目標、措施與活動使用人次統計（第 25 頁）。 |
| 「病歷完成率報表」與「未完成病歷名單」差別？ | 前者顯示完成率統計；後者列出未完成名單（第 27–28 頁）。 |
| 為什麼查不到約束報表？ | 須選擇護理站與時間後才能查詢「身體約束月報表」或「護理部月報表」（第 28–29 頁）。 |

---

## 補充說明
- 「管理作業」模組僅限具管理者或主管權限帳號使用。  
- 報表查詢結果可列印或下載，**目前僅支援 Excel 格式輸出**。  
- 「暫存」內容不會被其他模組引用；僅「完成」狀態可被正式使用。  
- 若畫面無反應或報表無法顯示，請重新登入並確認瀏覽器允許彈出視窗與下載。
