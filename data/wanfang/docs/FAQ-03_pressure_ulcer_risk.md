| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「壓瘡危險因子評估」後沒有任何資料？ | 需先選取護理站與病人後，點選「護理評估 → 壓瘡危險因子評估」，才能進入維護功能（第 3 頁）。 |
| 如何新增壓瘡危險因子評估？ | 點選【新增評估】，系統會開啟新增畫面；可修改預設的評估日期時間並展開各項目後進行填寫（第 3–4 頁）。 |
| 評估項目太多要怎麼展開或收合？ | 點擊每個項目前的資料夾圖示，即可展開或收合該項目的詳細說明（第 3 頁）。 |
| 存檔後沒有跳回首頁？ | 存檔完成後系統會自動返回首頁並顯示新增資料；若選擇【放棄】，則不儲存任何內容（第 4 頁）。 |
| 壓瘡危險因子評估分數低於 18 分會怎樣？ | 若總分 ≤18 且病人未發生壓瘡，系統會檢查是否已有健康問題「潛在危險性皮膚完整性受損」，若沒有會自動導向「新增健康問題作業」畫面（第 4 頁）。 |
| 病人已出現壓瘡時，新增評估後會發生什麼？ | 系統會跳出訊息「是否繼續填寫傷口護理記錄表」，按【確定】進入「傷口護理記錄首頁」，或按【取消】返回評估首頁（第 4 頁）。 |
| 新增評估後資料會去哪裡？ | 系統會自動將評估內容轉入護理記錄中（第 5 頁）。 |
| 修改評估內容後沒有更新？ | 點選【更改】進入修改畫面後，修改內容並按【存檔】即可更新；若按【放棄】則不會儲存（第 5 頁）。 |
| 修改後護理記錄會自動更新嗎？ | 會，系統會自動同步更新原評估資料所產生的護理記錄（第 5 頁）。 |
| 刪除評估時系統提示「是否確定刪除」？ | 按【確定】後系統才會刪除該筆資料；按【取消】則返回首頁（第 6 頁）。 |
| 為什麼刪除評估後護理記錄還在？ | 系統不會自動刪除相關護理記錄，需至護理記錄作業中手動刪除（第 6 頁）。 |
| 列印壓瘡危險因子評估表時沒有資料？ | 需先設定「列印日期區間」後按【查詢】，系統才會顯示預覽畫面（第 6–7 頁）。 |
| 預覽畫面沒反應或無法開啟？ | 請確認瀏覽器允許彈出視窗，列印前可點【預覽列印】或【列印】（第 7 頁）。 |
| 列印後想關閉畫面要怎麼做？ | 點選【離開】即可回到壓瘡危險因子評估首頁（第 7 頁）。 |

## 補充說明
- FAQ 內容皆依原《壓瘡危險因子評估操作手冊》頁碼整理。
