| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「疼痛評估」畫面後沒有病人資料？ | 需先於系統選取護理站與病人後，進入「生理監測 → 疼痛評估」畫面（第 3 頁）。 |
| 新增疼痛分數後跳出「請填寫初始疼痛評估表」訊息？ | 表示該病人尚未建立初始疼痛評估，依提示點【確定】進入初始疼痛評估畫面（第 4 頁）。 |
| 初始疼痛評估要怎麼存檔？ | 完成填寫後按【完成並設為正式摘要】，系統即將內容轉入護理記錄（第 7 頁）。 |
| 修改初始疼痛評估後沒有更新？ | 修改後需再次按【完成並設為正式摘要】儲存，系統才會同步更新（第 8 頁）。 |
| 新增持續疼痛評估時出現提示「無疼痛項目」？ | 若病人未建立「急性／慢性疼痛」健康問題，系統會自動開啟新增健康問題視窗（第 12 頁）。 |
| 想新增持續疼痛評估要怎麼做？ | 點【新增疼痛評估】，選擇疼痛類型與疼痛部位後輸入分數與描述，再按【存檔】（第 9–10 頁）。 |
| 為什麼疼痛分數無法輸入？ | 需先選擇疼痛量表（例如數字量表、臉譜量表、困難評估），系統才會開啟輸入欄位（第 3 頁）。 |
| 為什麼存檔後沒有跳出訊息？ | 存檔成功後會顯示結果訊息；若無反應請確認欄位是否填寫完整（第 10 頁）。 |
| 評估後要如何編輯疼痛部位？ | 於疼痛部位清單中選【修改】，更新內容後按【存檔】，系統會覆蓋原資料（第 10–11 頁）。 |
| 結案疼痛部位時出現提示「是否新增新部位」？ | 若結案原因非「疼痛解除」，系統會詢問是否新增新的疼痛部位（第 11 頁）。 |
| 為什麼刪除疼痛評估會出現「是否刪除相關護理記錄」？ | 按【確定】會連同護理記錄一起刪除；按【取消】則僅刪除該筆評估資料（第 15 頁）。 |
| 想列印疼痛評估表怎麼操作？ | 於查詢畫面設定「列印日期區間」與「編號」後，按【查詢】→【預覽列印】→【列印】（第 15 頁）。 |
| 列印預覽沒反應？ | 請確認瀏覽器允許彈出視窗後再執行【預覽列印】（第 15 頁）。 |
| 為什麼列印畫面沒顯示資料？ | 需輸入有效查詢條件（日期區間、編號）才會顯示疼痛紀錄（第 15 頁）。 |
| 刪除疼痛部位後相關記錄還存在？ | 系統不會自動刪除所有關聯護理記錄，需手動至「護理記錄」模組刪除（第 15 頁）。 |

---

## 補充說明
- 「疼痛評估」包含初始與持續兩種模式，**初始評估須先完成**，後續評估才可新增。  
- 若病人無相關健康問題或衛教項目，系統會自動開啟新增視窗。  
- 所有疼痛評估與健康問題、護理記錄皆自動連動更新。  
- 列印與預覽功能需允許瀏覽器彈出視窗；若無反應請重新登入或確認設定。
