| 常見問題 | 解決方法 |
|---|---|
| 為什麼進入「護理計劃」畫面後沒有看到任何健康問題？ | 需先選取護理站與病人後再進入「護理計劃」。系統預設僅顯示「未結束」問題，必要時可切換至「全部」清單（第 4 頁）。 |
| 點選「新增健康問題」後沒有出現任何項目？ | 在「選擇健康問題項目」畫面中，需依 NANDA 分類、科別、病房或診斷碼查詢後再選擇（第 6 頁）。 |
| 新增健康問題時無法存檔或出現錯誤提示？ | 新增時所有欄位須至少勾選一項；若護理目標文字欄位為空，必須填寫才能儲存（第 9 頁）。 |
| 為什麼新增的健康問題沒有顯示在清單？ | 需按【確定新增】完成寫入；若選【重新修改】或【放棄】，資料不會儲存（第 9 頁）。 |
| 修改定義性特徵後沒有更新？ | 按【存檔】後系統才會同步更新內容至護理記錄（第 10 頁）。 |
| 想結束某個定義性特徵，要怎麼操作？ | 在「修改定義性特徵」畫面取消勾選該特徵後按【存檔】即可結束（第 10 頁）。 |
| 相關因素或危險因子要如何新增？ | 於「修改相關因素／危險因子」畫面勾選未選項或使用【增加下一個相關因素／危險因子】新增（第 11 頁）。 |
| 新增護理目標後出現「未填寫欄位」錯誤？ | 護理目標文字欄位為必填；未填將無法通過存檔（第 12–14 頁）。 |
| 評值結果選「已達成」但目標仍未結束？ | 需正確存檔；當結果為「已達成」、「不適用」或「出院已衛教」時，系統會自動設定結束時間並關閉相關措施（第 15–16 頁）。 |
| 為什麼無法結束健康問題？ | 必須先透過「目標評值」將所有未結束的護理目標結束，才能結束整體健康問題（第 26 頁 注意事項 2）。 |
| 想刪除健康問題卻刪不掉？ | 僅能刪除「未結束」的健康問題，且「刪除原因」欄為必填。確認後按【刪除健康問題存檔】（第 21 頁）。 |
| 刪除健康問題後，相關護理記錄仍出現在系統？ | 系統不會自動刪除關聯的護理記錄，需手動至「護理記錄作業」中刪除（第 21 頁）。 |
| 出院通知後健康問題仍顯示？ | 出院通知僅結束該次住院的健康問題，不會刪除歷史記錄（第 22 頁）。 |
| 為什麼列印出的健康問題表沒有被刪除的項目？ | 系統列印時會自動排除已刪除項目，只顯示有效資料（第 23 頁）。 |
| 為什麼有些健康問題沒有「相關因素」欄位？ | 因健康問題類型不同：現存性含「定義性特徵＋相關因素」；潛在危險性僅有「危險因子」；健康促進型僅有「定義性特徵」（第 26 頁 注意事項 1）。 |

## 補充說明
- FAQ 依《護理計劃操作手冊》原頁碼整理。  
- 新增、刪除與結束動作皆需完整填寫必填欄位後方可存檔。  
- 出院通知僅結案，不影響歷史紀錄。  
- 若修改未更新，請重新整理頁面或確認權限設定。
