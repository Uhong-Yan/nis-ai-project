| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「給藥紀錄」畫面後沒有任何內容？ | 需先選擇護理站與病人後，點選「給藥紀錄」作業項目，系統才會顯示功能清單（第 3 頁）。 |
| 「執行給藥紀錄」查不到待執行的藥囑？ | 請確認查詢區間設定正確；系統預設區間為系統時間前 8 小時至後 1 小時（第 4 頁）。 |
| 超過上午 10 點後查不到前一日的紀錄？ | 10 點前查詢區間為前一日 0 時至系統時間後 1 小時；10 點後僅能查當日 0 時起的紀錄（第 4 頁）。 |
| 為什麼某些藥品標示紅色、黑色或粉紅色？ | 紅色：逾期 30 分鐘未執行；黑色：在標準執行區間內；粉紅色：尚未到執行時間（第 4 頁）。 |
| 想看藥品詳細資訊要去哪裡？ | 點選藥囑名稱前的小藥典圖示，可開啟藥品說明視窗（第 4 頁）。 |
| 為什麼完成給藥後畫面沒更新？ | 點選待執行藥品後按【完成】→【確認】存檔，系統才會更新主畫面（第 5 頁）。 |
| 可以修改給藥執行時間嗎？ | 若該筆記錄顯示「執行日期時間」欄，可自行調整；若無此欄位，系統以完成時間為主（第 5 頁）。 |
| PRN 藥為什麼出現時間提示？ | 系統會檢查前次至本次間隔是否異常，例如 Q4H PRN 若未滿 4 小時會提示，但不限制執行（第 5 頁）。 |
| 想標註「延遲」或「未執行」要怎麼操作？ | 點選紀錄後方的按鈕，開啟「未執行原因選擇」選單，選取原因後按【完成】存檔（第 5–6 頁）。 |
| 為什麼有的原因顯示藍色、有的顯示紅色？ | 延遲原因以藍色標示；未執行原因以紅色標示（第 6 頁）。 |
| 可以一次設定多筆延遲原因嗎？ | 常規醫囑可使用【批次原因註記】批次設定延遲或未執行原因（第 6–7 頁）。 |
| 想復原修改後的資料？ | 點選【復原】按鈕，可重新載入最後存檔狀態（第 7 頁）。 |
| 怎麼取消錯誤的給藥紀錄？ | 進入「取消給藥紀錄」，選擇時間區間查詢後勾選欲取消的紀錄，按【完成】即可（第 8–9 頁）。 |
| 可以一次取消多筆紀錄嗎？ | 使用【全選】可批次取消所有顯示的已執行紀錄（第 9 頁）。 |
| 為什麼查詢紀錄頁面沒有資料？ | 需輸入查詢區間後按【查詢】，系統才會列出該時段所有給藥紀錄（第 10 頁）。 |
| 想列印特定時段的給藥紀錄？ | 進入「給藥紀錄列印」，輸入查詢區間後按【查詢】→【預覽列印】→【Print】（第 10–11 頁）。 |
| 點選列印沒反應？ | 請確認瀏覽器允許彈出視窗，列印頁面中點【Print】即可執行輸出（第 11 頁）。 |

---

## 補充說明
- 「給藥紀錄」屬於高權限模組，部分功能（如批次註記、取消作業）僅限具授權帳號使用。  
- 所有操作均需完成【確認】或【存檔】步驟，否則系統不會更新資料。  
- 若畫面未更新或按鈕無反應，請重新登入並確認網路連線與瀏覽器彈出設定。  
- 列印僅支援預覽頁面輸出，若需匯出報表可透過管理者帳號執行。
