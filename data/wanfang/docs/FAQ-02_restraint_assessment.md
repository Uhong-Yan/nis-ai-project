| 常見問題 | 解決方法 |
|---|---|
| 為什麼進入「約束評估記錄」後沒有資料？ | 需先選取護理站與病人後，再進入「護理評估 → 約束評估記錄」功能（第 3 頁）。 |
| 新增約束時無法輸入開始時間？ | 系統預設「開始日期時間」為目前系統時間，可修改但不可留空；完成後按【確定】（第 3 頁）。 |
| 新增約束後資料沒有出現在清單？ | 新增成功後內容會自動轉入護理記錄，若未顯示可重新查詢（第 4 頁）。 |
| 為什麼無法新增評估記錄？ | 須先新增約束才能進行新增評估記錄作業（第 11 頁 注意事項 2）。 |
| 想設定特殊時間點的評估要怎麼做？ | 點選【特殊狀況評估】可自行修改評估日期時間，但相同時間點不可重複存檔（第 5 頁）。 |
| 存檔時出現「評估記錄未填寫」錯誤？ | 新增評估記錄時所有內容須填寫或至少勾選一項才能存檔（第 5 頁）。 |
| 評估內容若出現「皮膚破損」會發生什麼事？ | 系統會提示「是否進入傷口護理記錄作業？」可直接開啟傷口模組填寫（第 5 頁）。 |
| 為什麼有些評估記錄沒進護理記錄？ | 系統僅將「第一次評估」及每日 09:00、17:00、01:00 評估內容轉入護理記錄（第 5 頁）。 |
| 結束約束時出現「尚有評估記錄未填寫！」？ | 表示仍有常規約束評估未完成，需先回首頁補寫所有評估記錄後才能結束（第 6 頁）。 |
| 結束約束後資料沒進護理記錄？ | 完成結束後系統會自動將內容轉入護理記錄，若未顯示可重新登入查詢（第 7 頁）。 |
| 刪除約束時出現「已有評估記錄無法刪除」？ | 若該次約束已有評估資料，需先刪除所有評估記錄後才能刪除該約束（第 7 頁）。 |
| 刪除約束後仍顯示舊資料？ | 系統僅刪除該次約束內容，若欲清除相關護理記錄須至護理記錄作業中手動刪除（第 9 頁）。 |
| 修改評估記錄後護理記錄沒有更新？ | 修改後系統會自動同步更新相關護理記錄；若刪除評估，需自行決定是否刪除護理記錄（第 9 頁）。 |
| 列印約束評估記錄表時沒顯示內容？ | 請先設定「列印日期區間」與「約束次數」後按【查詢】，系統才會顯示預覽畫面（第 9 頁）。 |
| 為什麼無法列印？ | 僅完成查詢後的資料可列印；確認已輸入查詢條件後按【列印】（第 9–10 頁）。 |
| 「結束日期時間」設定錯誤導致無法存檔？ | 結束時間不可小於開始時間，且不得早於該約束內任何評估時間（第 11 頁）。 |
| 為什麼系統顯示「約束時間早於前次結束時間」？ | 每次約束的開始時間不可小於上一次約束的結束時間；若為第一次約束則不可小於入院日期（第 11 頁）。 |
| 可以修改已結束的約束嗎？ | 不可直接修改結束時間，若需更改，須刪除原結束時間後重新編輯（第 7 頁）。 |
| 列印畫面開啟後無法輸出？ | 請確認瀏覽器允許彈出視窗，點選【列印】即可送出列印作業（第 10 頁）。 |
| 為什麼「日期時間」僅限制不能大於系統時間？ | 系統為確保資料正確，禁止未來時間輸入，所有時間須小於或等於系統時間（第 11 頁）。 |
