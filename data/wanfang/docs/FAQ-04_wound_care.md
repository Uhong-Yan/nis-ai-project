| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「傷口護理記錄」畫面後沒有病人資料？ | 需先在登入系統選取護理站與病人後，點選「護理記錄 → 傷口護理記錄」才能進入維護功能（第 3 頁）。 |
| 新增傷口時按【確定】沒有反應？ | 檢查是否已填寫「發生日期」、選擇「傷口種類」及「部位」；若未選擇任何項目，系統不會執行新增（第 3–4 頁）。 |
| 為什麼新增傷口後系統自動跳出視窗？ | 新增存檔成功後系統會詢問「是否新增傷口記錄？」；按【確定】會直接開啟新增記錄畫面（第 4 頁）。 |
| 選擇「壓瘡」類型時出現提示？ | 當傷口種類為「壓瘡」時，必須輸入「發生地點」與「發生原因」，否則無法存檔（第 4 頁）。 |
| 為什麼「發生日期」輸入後出錯？ | 系統限制「發生日期」不得大於目前系統時間；若不清楚可勾選「不詳」（第 4 頁）。 |
| 新增傷口記錄時存檔失敗？ | 所有欄位內容皆須填寫或至少勾選一項，否則系統不允許存檔（第 6 頁）。 |
| 新增記錄後系統自動跳轉到新增健康問題畫面？ | 若該病人現存健康問題中沒有「皮膚完整性受損」，系統會自動導向「新增健康問題作業」畫面（第 6 頁）。 |
| 為什麼新增記錄後沒有自動產生衛教項目？ | 只有當勾選的護理措施為「皮膚癢處理衛教」、「疤痕處理衛教」或「居家照護衛教」時，系統才會自動新增衛教項目（第 6 頁）。 |
| 更改或刪除記錄後內容沒有更新？ | 需按【記錄更改存檔】完成操作；若只是刪除，系統不會自動移除相關護理記錄，需人工刪除（第 7 頁）。 |
| 傷口結案後又跳出「是否新增傷口？」提示？ | 當「結案原因」不是「痊癒」時，系統會詢問是否要新增新的傷口資料（第 8 頁）。 |
| 為什麼刪除傷口後仍看到舊資料？ | 系統不會同步刪除所有護理記錄，需人工至「護理記錄作業」中刪除相關資料（第 9 頁）。 |
| 刪除傷口時出現「是否刪除相關護理記錄」訊息？ | 按【確定】會同時刪除該傷口下所有記錄；按【取消】則僅刪除傷口基本資料（第 9 頁）。 |
| 為什麼列印傷口護理記錄表沒有資料？ | 需先輸入「列印日期區間」、「傷口種類」與「傷口編號」後點選【查詢】才會生成預覽（第 10 頁）。 |
| 列印後畫面卡住或無反應？ | 請確認瀏覽器允許彈出視窗；可使用【離開】關閉預覽畫面返回首頁再嘗試列印（第 10 頁）。 |
| 為什麼無法選擇未結案傷口進行刪除？ | 僅未結束的傷口資料可進行刪除或修改；已結案者無法再刪除（第 7–8 頁）。 |

## 補充說明
- FAQ 內容依原《傷口護理記錄操作手冊》頁碼編製。  
- 若操作無反應，請確認所有必要欄位已填寫且瀏覽器未封鎖彈出視窗。  
- 欲刪除相關護理記錄，請使用「護理記錄作業」模組手動清除。
