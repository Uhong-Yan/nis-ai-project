| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「護理記錄」畫面後沒有任何資料？ | 需先選取護理站與病人後，再點選「護理記錄」進入維護功能；可選【一般護理記錄】或【傷口護理記錄】（第 3 頁）。 |
| 查詢護理記錄時為什麼沒顯示完整內容？ | 查詢區間預設為「系統前兩日至目前時間」，可手動設定日期區間後按【查詢】（第 3 頁）。 |
| 為什麼部分記錄顯示紅色？ | 紅色代表該筆護理記錄不可修改，只能補述內容（第 3 頁）。 |
| 想修改護理記錄要怎麼操作？ | 點選明細右側【更改】，修改後按【護理記錄更新存檔】，再於確認視窗按【確定】（第 4 頁）。 |
| 修改後畫面沒更新？ | 存檔完成後系統會自動返回清單並顯示更新後資料（第 4 頁）。 |
| 刪除護理記錄後仍顯示？ | 刪除後需在確認視窗中按【確定】，系統才會真正刪除該筆資料（第 5 頁）。 |
| 為什麼列印護理記錄出現換頁錯誤？ | 系統預設【否】不換頁；若印表機具雙面功能請選【可】，再依提示設定「是否換頁」及「上次列印位置」（第 6 頁）。 |
| 列印後右邊出現一組號碼是什麼？ | 為「上次列印位置」，用於下次續印（第 7 頁）。 |
| 新增護理記錄時可以插入圖表嗎？ | 可點【EKG】插入約 3.6 cm 高度的空白行，用於放置心電圖影像（第 9 頁）。 |
| 想快速輸入常用內容要怎麼做？ | 點【片語】進入清單，依分類勾選後系統會自動帶入對應內容（第 9 頁）。 |
| 片語帶入功能沒反應？ | 確認「管理作業 → 片語維護」中該片語已設為【完成】；暫存狀態無法使用（第 13 頁）。 |
| 如何新增片語？ | 於片語目錄按【新增片語】，輸入名稱、編號與內容後選【完成】，即顯示於清單（第 12–13 頁）。 |
| 暫存與完成片語差別？ | 【暫存】僅暫時保存、不可使用；【完成】代表可正式使用（第 13 頁）。 |
| 想刪除片語卻沒刪除成功？ | 點【刪】後系統會跳出確認訊息，按【確定】後才會刪除（第 15 頁）。 |
| 如何建立新的片語分類？ | 在左側目錄選定位置後，按【建立新的目錄】→ 輸入名稱與說明 → 【確定】（第 15 頁）。 |
| 想改片語目錄名稱？ | 選取目錄後按【目錄重新命名】，修改名稱與說明後按【確定】（第 16 頁）。 |
| 目錄移動後沒有更新？ | 點【移動目錄位置】選新位置後按【確定】，系統會在目錄區顯示更新後位置（第 17 頁）。 |
| 刪除目錄後整個子目錄都不見？ | 系統會連同子目錄與底下片語一併刪除，操作前請確認（第 18 頁）。 |
| 為什麼刪除片語後無法回復？ | 刪除為不可逆操作，執行後系統即永久移除該資料（第 19 頁）。 |

---

## 補充說明
- 護理記錄可查詢、修改、刪除，但部分系統自動產生紀錄（如護理計畫或傷口記錄）僅能檢視。  
- 「片語維護」與「目錄管理」屬管理作業模組，需有相應權限才能使用。  
- 片語僅在設定為【完成】狀態時可供選用，暫存版本不會出現在清單。  
- 刪除片語與目錄為不可復原操作；建議管理者操作前再次確認。
