| 常見問題 | 解決方法 |
|---|---|
| 為什麼打開系統後無法登入？ | 確認網址正確並使用 IE 瀏覽器開啟新視窗後輸入帳號與密碼，再按【確定】登入（第 3 頁）。 |
| 登入後畫面空白或沒反應？ | 檢查網路連線與網址正確性；登入成功後應出現主畫面（第 3 頁）。 |
| 為什麼出現「帳號錯誤」訊息？ | 確認帳號密碼是否正確；若多次錯誤導致鎖定，請聯絡系統管理員重設密碼（第 3 頁）。 |
| 系統自動登出是正常的嗎？ | 是。為確保資料安全，若閒置過久或關閉瀏覽器，系統會自動登出（第 6 頁 注意事項 4–5）。 |
| 忘記登出會有影響嗎？ | 可能導致他人以你的帳號操作，請養成離開電腦時立即登出的習慣（第 6 頁 注意事項 4）。 |
| 如何安全登出系統？ | 點選畫面右上角的【登出】即可安全離開系統（第 6 頁 注意事項 5）。 |
| 為什麼無法選擇病人？ | 必須先登入並選擇護理站後，系統才會顯示該站病人清單（第 4 頁）。 |
| 病人清單載不出來？ | 確認已選擇正確護理站；若仍無顯示，可重新登入或確認網路狀況（第 4 頁）。 |
| 病人清單排序混亂？ | 系統依「床號順序」自動排序顯示（第 4 頁 步驟 3）。 |
| 點選病人後沒跳出資料？ | 點選病人後系統會自動帶出該次住院資料，若未顯示請重新登入或檢查病歷（第 4 頁）。 |
| 想用病歷號查病人？ | 於輸入框輸入病歷號後按【查】，若有多次住院會列出清單供選擇（第 5 頁）。 |
| 為什麼無法輸入前一天的資料？ | 每日上午 10 點前可輸入前一日資料；10 點後僅可輸入系統時間前 12 小時內資料（第 6 頁 注意事項 1）。 |
| 想補輸過去資料卻無法？ | 僅主管或護理長級帳號可勾選【當機補輸資料】以補登；一般帳號無法使用（第 6 頁 注意事項 2–3）。 |
| 「當機補輸資料」選項在哪裡？ | 登入主管帳號後，在記錄畫面中可見【當機補輸資料】核取方塊（第 6 頁）。 |
| 為什麼沒有「當機補輸資料」選項？ | 一般帳號無此權限，需主管授權（第 6 頁）。 |
| 護理站或病人清單載入很慢？ | 請確認網路穩定並使用 IE 瀏覽器；避免同時多重登入（第 3–4 頁）。 |

## 補充說明
- FAQ 內容皆依據《登入系統操作手冊》對照頁碼整理。
