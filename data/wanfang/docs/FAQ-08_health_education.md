| 常見問題 | 解決方法 |
|---|---|
| 為什麼開啟「衛教記錄」畫面後沒有資料？ | 需先選取護理站與病人後，才能進入「衛教記錄」維護功能（第 3 頁）。 |
| 衛教清單只顯示部分項目？ | 系統預設僅顯示「未評值」項目，可點清單抬頭切換成「全部」查看（第 3 頁）。 |
| 想查特定人員的衛教紀錄怎麼做？ | 於首頁「欲查詢之職稱」選單中選取職稱，系統即顯示該職稱的所有衛教項目（第 3 頁）。 |
| 「說明內容」欄位的按鈕不見？ | 僅部分衛教項目具電子檔單張，若暫無電子資料，按鈕不會顯示（第 4 頁）。 |
| 新增衛教項目時查不到需要的主題？ | 在「選擇衛教項目」畫面中，可依診斷碼、關鍵字、分類或「需加強項目」查詢；如仍無資料，可用【增加其他衛教項目】自行新增（第 4–5 頁）。 |
| 查詢結果中為什麼找不到曾經教過的項目？ | 系統會排除已執行過的衛教項目，因此不再出現在新增清單（第 4 頁）。 |
| 執行衛教時無法存檔？ | 存檔時所有內容必須至少勾選一個項目；未勾選系統不允許儲存（第 7 頁）。 |
| 執行衛教存檔後系統跳出「是否進入衛教評值？」訊息？ | 表示存檔成功。點【確定】可直接進入「衛教評值」畫面，點【取消】則返回首頁（第 7 頁）。 |
| 衛教評值後還顯示「部分了解」或「需再加強」怎麼辦？ | 存檔後系統會詢問「是否再進入執行衛教」以加強教學，可按【確定】重新執行（第 8 頁）。 |
| 為什麼無法進行衛教評值？ | 需先在首頁勾選要評值的項目（可複選）後，點【衛教評值】進入畫面（第 8 頁）。 |
| 衛教評值的日期怎麼修改？ | 系統預設為目前日期時間，可自行調整（第 8 頁）。 |
| 評值後系統沒有更新護理記錄？ | 必須按【衛教評值存檔】，成功後系統才會把結果轉入護理記錄（第 8 頁）。 |
| 衛教單列印時為什麼缺資料？ | 系統僅列出「已執行」的衛教項目，未執行的不會顯示（第 9 頁）。 |
| 要怎麼一次列印所有病人衛教內容？ | 使用【衛教單列印】功能即可輸出該病人自入院至目前所有已執行衛教項目（第 9 頁）。 |

## 補充說明
- FAQ 內容依《衛教記錄操作手冊》頁碼整理。  
- 未執行或未評值項目不會出現在列印清單中。  
- 儲存與評值完成後系統會自動同步至「護理記錄」模組。  
- 若畫面或列印按鈕無反應，請重新登入或確認瀏覽器未封鎖彈出視窗。
